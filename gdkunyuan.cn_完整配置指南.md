# gdkunyuan.cn 域名配置完整指南

## 当前状态

域名 `gdkunyuan.cn` 已购买，需要完成以下步骤：

1. 实名认证（1-3 小时到 1 天）
2. 域名备案（7-20 天）
3. CDN 配置
4. DNS 解析
5. 验证访问

---

## 第一步：确认实名认证状态

### 检查实名认证

1. 打开腾讯云域名控制台
   👉 https://console.cloud.tencent.com/domain

2. 找到 `gdkunyuan.cn`

3. 查看 **「实名认证」** 状态：
   - ✅ **已认证** → 可以继续配置 DNS
   - ⏳ **审核中** → 需要等待 1-3 小时到 1 天
   - ❌ **未认证** → 需要先提交认证

### 如果需要实名认证

1. 点击 **「实名认证」**

2. 选择认证方式：
   - **个人认证**：上传身份证照片
   - **企业认证**：上传营业执照和法人身份证

3. 等待审核（通常 1-3 小时）

---

## 第二步：实名认证通过后，配置 CDN

### 在腾讯云 CDN 添加域名

1. 打开 CDN 控制台
   👉 https://console.cloud.tencent.com/cdn

2. 点击 **「添加域名」**

3. 填写基本信息：
   - **加速域名**：`www.gdkunyuan.cn`
   - **业务类型**：网页加速
   - **服务区域**：中国大陆

4. 配置源站：
   - **源站类型**：选择 **COS 源**
   - **源站地址**：从下拉列表选择
     - 存储桶：`kunyuan-1257542161`
     - 地域：`ap-guangzhou`（广州）

5. 其他配置（使用默认值）：
   - **回源协议**：HTTP
   - **回源 Host**：留空

6. 点击 **「提交」**

---

## 第三步：获取 CNAME 地址

### 查看 CNAME

1. 在 CDN 控制台 → **「域名管理」**

2. 找到 `www.gdkunyuan.cn`

3. 复制 **CNAME 地址**（类似：`www.gdkunyuan.cn.cdn.dnsv1.com`）

---

## 第四步：配置 DNS 解析

### 在 DNSPod 添加 CNAME 解析

1. 打开 DNSPod 控制台
   👉 https://console.cloud.tencent.com/cns

2. 找到域名 `gdkunyuan.cn`

3. 点击 **「解析」**

4. 添加记录：

| 主机记录 | 记录类型 | 线路类型 | 记录值 | TTL |
|---------|---------|---------|-------|-----|
| `www` | `CNAME` | 默认 | CDN 提供的 CNAME 地址 | 600 |

5. 点击 **「保存」**

---

## 第五步：等待 CDN 生效

### 等待时间

- **DNS 生效**：10-30 分钟
- **CDN 生效**：5-10 分钟

### 验证 CDN 生效

访问：
```
https://www.gdkunyuan.cn
```

**如果配置正确，网站应该正常显示！**

---

## 重要提醒：域名备案

### .cn 域名必须备案

**注意：** `.cn` 域名必须备案后才能在中国大陆访问。

### 备案流程

1. 打开腾讯云备案系统
   👉 https://console.cloud.tencent.com/beian

2. 点击 **「开始备案」**

3. 填写主体信息：
   - **主体类型**：个人/企业
   - **证件类型**：身份证/营业执照
   - **证件号码**：您的证件号
   - **联系人信息**：您的联系方式

4. 填写网站信息：
   - **网站名称**：广东坤元
   - **网站域名**：`gdkunyuan.cn`
   - **网站内容**：企业官网

5. 上传资料：
   - 主体证件照片
   - 网站备案信息表
   - 负责人身份证

6. 提交审核：
   - **腾讯云审核**：1-2 天
   - **管局审核**：7-20 天

### 备案期间

**网站无法在中国大陆访问**，但可以：
- ✅ 海外访问正常
- ✅ 测试环境可以访问
- ❌ 中国大陆用户无法访问

## 临时解决方案：Vercel（无需备案）

如果您不想等备案，可以先部署到 Vercel：

### 部署到 Vercel

1. 打开 👉 https://vercel.com

2. 注册/登录账号

3. 点击 **「Add New」** → **「Project」**

4. 选择 **「Upload」**

5. 拖拽文件夹：
   ```
   C:\Users\oucx\WorkBuddy\20260320112509\kunyuan-web
   ```
   （先删除 `node_modules` 和 `dist`）

6. 点击 **「Deploy」**

7. 使用 Vercel 提供的域名：
   ```
   https://kunyuan-web-xxx.vercel.app
   ```

### 绑定域名（无需备案）

1. 在 Vercel → **「Domains」** → **「Add Domain」**

2. 输入 `gdkunyuan.cn`

3. 在 DNSPod 添加 CNAME 解析：
   - 主机记录：`@`（根域名）
   - 记录类型：`CNAME`
   - 记录值：Vercel 提供的地址（类似 `cname.vercel-dns.com`）

4. 完成！

**这样配置后，不需要备案也能访问！**

## 快速选择

### 选项 A：等备案通过后使用 CDN（长期稳定）

**优势：**
- 国内访问更快
- 符合国内法规
- 更稳定

**缺点：**
- 需要等 7-20 天
- 备案期间无法访问

**适合：**
- 有充足时间等待
- 主要面向国内用户

---

### 选项 B：立即部署到 Vercel（快速上线）

**优势：**
- 立即上线（5 分钟）
- 无需备案
- 完全免费
- 全球 CDN 加速

**缺点：**
- 国内访问速度稍慢
- 不符合国内法规（长期）

**适合：**
- 需要立即上线
- 测试阶段
- 面向海外用户

---

### 选项 C：Vercel + CDN 双轨并行（推荐）

**优势：**
- 立即有网站可用（Vercel）
- 同时准备备案
- 备案后切换到 CDN

**缺点：**
- 需要配置两次

**适合：**
- 既想立即上线
- 又要长期稳定
- 所有场景
