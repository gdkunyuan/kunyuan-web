## 📋 接下来需要您完成的步骤

### 步骤 1：在 GitHub 创建新仓库

1. 访问：https://github.com/new
2. 仓库名（Repository name）：`kunyuan-web`
3. 描述（Description，可选）：`广东坤元企业官网`
4. 选择 **Public**（公开）或 **Private**（私有）
5. **不要**勾选 "Add a README file"
6. 点击 **Create repository**

### 步骤 2：创建 GitHub Personal Access Token

因为 Git 推送需要使用 Token，而不是密码：

1. 访问：https://github.com/settings/tokens
2. 点击 **Generate new token** → **Generate new token (classic)**
3. Note：填写 `Vercel Deploy`
4. Expiration：选择 **90 days**
5. 勾选权限：✅ **repo**（完整的仓库控制权限）
6. 点击 **Generate token**
7. **复制 Token**（只显示一次，请妥善保存！）

### 步骤 3：告诉我 Token 已创建

创建完成后，请告诉我："Token 已创建"，然后我会帮您执行推送命令。

推送时会提示输入密码，您需要：
- **用户名**：`gdkunyuan`
- **密码**：粘贴刚才创建的 Token
