# 腾讯云 CDN 开通完整步骤

## 问题原因
当前账号未开通 CDN 服务，无法通过 API 添加域名。

---

## 解决方案：手动开通 CDN

### 第一步：开通 CDN 服务

1. 打开浏览器访问：
   ```
   https://console.cloud.tencent.com/cdn
   ```

2. 如果提示「开通 CDN 服务」或「免费开通」，点击 **「开通 CDN」**

3. 开通后，控制台会自动跳转到 CDN 域名管理页面。

---

### 第二步：添加加速域名

1. 点击 **「添加域名」**

2. 填写以下配置：

| 配置项 | 填写值 |
|--------|--------|
| 加速域名 | `kunyuan-1257542161.cos.ap-guangzhou.myqcloud.com` |
| 业务类型 | 静态加速 |
| 源站类型 | **COS 源** |
| 源站地址 | 自动选择 `kunyuan-1257542161`（广州） |
| 服务区域 | 中国大陆 |

3. 点击 **「提交」**

---

### 第三步：配置缓存规则（重要）

域名添加成功后：

1. 点击域名进入 **「域名管理」**

2. 进入 **「缓存配置」** 标签页

3. 添加以下缓存规则：

| 匹配类型 | 匹配内容 | 过期时间 |
|---------|---------|---------|
| 文件扩展名 | `html` | **0**（不缓存） |
| 文件扩展名 | `js,css` | 2592000 秒（30 天） |
| 文件扩展名 | `svg` | 604800 秒（7 天） |

> ⚠️ **HTML 不缓存非常重要**，否则更新内容后访问的还是旧版本。

---

### 第四步：（可选）配置 HTTPS

1. 进入 **「HTTPS 配置」** 标签页

2. 点击 **「修改配置」**

3. 选择证书来源：
   - 如果有域名，可以申请免费证书（腾讯云提供免费 DV 证书）
   - 临时可以先不配置，用 HTTP 访问

---

### 第五步：访问网站

CDN 生效需要 5–10 分钟，生效后访问：

```
http://kunyuan-1257542161.cos.ap-guangzhou.myqcloud.com
```

> 注意：是 `.cos.` 不是 `.cos-website.`

---

|------|------|
| `https://kunyuan-1257542161.cos-website.ap-guangzhou.myqcloud.com` | ❌ 会下载（Content-Type 问题） |
| `http://kunyuan-1257542161.cos.ap-guangzhou.myqcloud.com` | ✅ 正常访问（CDN 已修复） |

---

## 后续优化

开通 CDN 后，如果需要绑定自己的域名（如 `www.kunyuan.com`）：

1. 在 CDN 控制台添加加速域名 `www.kunyuan.com`
2. 源站选择 COS 存储桶
3. 域名注册商处添加 CNAME 解析（指向 CDN 提供的 CNAME 地址）
4. 配置 HTTPS 证书
