# 腾讯云 COS 静态网站部署指南

## 问题分析

您之前访问的是：
```
kunyuan-1257542161.cos-website.ap-guangzhou.myqcloud.com
```

这个域名出现"下载页面"的问题，是因为 **COS 静态网站托管** 没有正确配置索引页。

---

## 解决方案：正确配置 COS 静态网站托管

### 第一步：开启「静态网站托管」功能

1. 打开腾讯云 COS 控制台
   👉 https://console.cloud.tencent.com/cos

2. 进入存储桶：`kunyuan-1257542161`（广州）

3. 左侧菜单 → **「基础配置」** → 向下滚动找到 **「静态网站」**

4. 点击 **「编辑」** 或 **「开启」**

5. 配置静态网站：
   - ✅ **当前状态**：开启
   - **索引文档**：`index.html`
   - **错误文档**：`404.html`（可选）

6. 点击 **「保存」**

---

### 第二步：确认域名配置

#### 正确的静态网站域名（可正常访问）

```
kunyuan-1257542161.cos-website.ap-guangzhou.myqcloud.com
```

#### 错误的普通域名（会出现下载问题）

```
kunyuan-1257542161.cos.ap-guangzhou.myqcloud.com
```

> ⚠️ 必须使用 `.cos-website.` 域名，不是 `.cos.`

---

### 第三步：验证访问

开启静态网站托管后，等待 1-2 分钟，访问：

```
https://kunyuan-1257542161.cos-website.ap-guangzhou.myqcloud.com
```

**如果配置正确，网站应该正常显示，不会下载！**

---

### 第四步：如果还有问题

#### 问题：仍然下载页面

**原因**：Content-Type 头部设置错误

**解决方案**：
1. 在 COS 控制台 → **「基础配置」** → **「静态网站」**
2. 确认 **索引文档** 设置为 `index.html`
3. 如果有 HTTPS 证书问题，可以暂时关闭 HTTPS

---

### 第五步：使用自定义域名（推荐）

等 `gdkunyuan.cn` 备案通过后，绑定自定义域名：

#### 方式 A：在 COS 绑定自定义域名（简单）

1. 打开 COS 控制台 → **「域名管理」** → **「自定义域名」**
2. 点击 **「添加域名」**
3. 填写：
   - **域名**：`www.gdkunyuan.cn`
   - **源站**：选择静态网站域名
   - **访问类型**：静态网站
4. 在 DNSPod 添加 CNAME 解析

#### 方式 B：使用 CDN（专业，推荐）

1. 打开 CDN 控制台
   👉 https://console.cloud.tencent.com/cdn
2. 点击 **「添加域名」**
3. 填写：
   - **加速域名**：`www.gdkunyuan.cn`
   - **业务类型**：网页加速
4. **源站配置**：
   - **源站类型**：COS 源
   - **源站地址**：选择 `kunyuan-1257542161`（广州）
   - **回源协议**：HTTP
5. 在 DNSPod 添加 CNAME 解析

## 快速检查清单

- [ ] 存储桶已上传 `index.html` 到根目录
- [ ] 存储桶访问权限：**公共读**（Public Read）
- [ ] 静态网站已开启
- [ ] 索引文档设置为 `index.html`
- [ ] 访问的是 `.cos-website.` 域名
