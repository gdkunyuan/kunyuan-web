# 域名备案指南（7-20 天）

## 备案前准备

### 第一步：确认主体类型

#### 个人备案
- ✅ 适用于：个人博客、个人网站
- 📝 需要：身份证

#### 企业备案
- ✅ 适用于：企业官网、企业网站
- 📝 需要：营业执照 + 法人身份证

---

### 第二步：准备备案材料

#### 个人备案材料清单

| 材料 | 说明 | 格式要求 |
|------|------|---------|
| 身份证正面 | 身份证照片正面 | JPG/PNG，清晰 |
| 身份证反面 | 身份证照片反面 | JPG/PNG，清晰 |
| 手持身份证照片 | 手持身份证自拍 | JPG/PNG，清晰 |
| 网站备案信息表 | 腾讯云提供模板 | PDF |
| 备案承诺书 | 腾讯云提供模板 | PDF |
| 验证码接收 | 手机号 | 实名手机 |

#### 企业备案材料清单

| 材料 | 说明 | 格式要求 |
|------|------|---------|
| 营业执照 | 营业执照照片 | JPG/PNG，清晰 |
| 法人身份证 | 法人身份证正反面 | JPG/PNG，清晰 |
| 网站备案信息表 | 腾讯云提供模板 | PDF |
| 备案授权书 | 如需 | PDF |
| 验证码接收 | 手机号 | 实名手机 |

---

### 第三步：下载备案模板

#### 网站备案信息表

1. 打开腾讯云备案系统
   👉 https://console.cloud.tencent.com/beian

2. 点击 **「开始备案」**

3. 填写主体信息（下一步会下载模板）

4. 下载 **网站备案信息表**
   - 填写完整
   - 打印签字
   - 拍照或扫描

---

## 备案流程

### 第四步：提交备案申请

1. 打开腾讯云备案系统
   👉 https://console.cloud.tencent.com/beian

2. 点击 **「开始备案」**

3. 选择主体类型：
   - **个人备案** 或 **企业备案**

---

### 第五步：填写主体信息

#### 个人备案填写：

| 字段 | 填写内容 |
|------|---------|
| 主体类型 | 个人 |
| 证件类型 | 身份证 |
| 证件号码 | 您的身份证号 |
| 姓名 | 您的真实姓名 |
| 性别 | 您的性别 |
| 出生日期 | 您的出生日期 |
| 现居地址 | 您的详细地址（省/市/区/街道） |
| 手机号 | 您的手机号（用于接收验证码） |
| 电子邮箱 | 您的邮箱 |

#### 企业备案填写：

| 字段 | 填写内容 |
|------|---------|
| 主体类型 | 企业 |
| 证件类型 | 营业执照 |
| 证件号码 | 统一社会信用代码 |
| 企业名称 | 营业执照上的企业名称 |
| 法人姓名 | 法定代表人姓名 |
| 法人身份证号 | 法人身份证号 |
| 企业地址 | 营业执照上的地址 |
| 联系人姓名 | 联系人姓名 |
| 联系人手机号 | 联系人手机号 |
| 电子邮箱 | 企业邮箱 |

### 第七步：上传资料

#### 上传证件照片

1. 按要求上传：
   - 身份证正面照片
   - 身份证反面照片
   - 手持身份证照片
   - 或营业执照照片

2. 确保照片：
   - ✅ 清晰可见
   - ✅ 无反光
   - ✅ 无遮挡
   - ✅ 边框完整

#### 上传备案信息表

1. 上传填写的网站备案信息表
   - 已签字
   - 已打印或扫描

---

### 第八步：提交审核

1. 检查所有信息无误

2. 点击 **「提交审核」**

3. 等待腾讯云初审

---

## 审核流程

### 时间线

| 阶段 | 时间 | 说明 |
|------|------|------|
| **腾讯云初审** | 1-2 天 | 检查资料完整性 |
| **管局审核** | 7-20 天 | 最终审批 |
| **备案通过** | 总计 7-20 天 | 可以使用 CDN |

---

### 审核期间

#### ✅ 可以做的：
- 查询备案状态
- 修改备案资料
- 补充材料
- 海外访问（如果配置了）

#### ❌ 不能做的：
- 域名在中国大陆访问
- 使用腾讯云 CDN

---

### 备案状态查询

1. 打开腾讯云备案系统
   👉 https://console.cloud.tencent.com/beian

2. 查看备案状态：
   - **审核中**：正在审核
   - **审核不通过**：需要修改资料
   - **备案通过**：可以使用 CDN

---

## 备案通过后

### 第九步：配置 CDN

1. 打开 CDN 控制台
   👉 https://console.cloud.tencent.com/cdn

2. 添加域名：
   - 加速域名：`www.gdkunyuan.cn`
   - 源站类型：COS 源
   - 源站地址：`kunyuan-1257542161`

3. 获取 CNAME 地址

4. 配置 DNS 解析

---

### 第十步：切换到腾讯云 CDN

#### 从 Vercel 切换到 CDN

1. 在 Vercel 删除域名绑定（保留项目）

2. 在 DNSPod 修改 CNAME：
   - 主机记录：`www`
   - 记录类型：`CNAME`
   - 记录值：CDN 提供的 CNAME 地址

3. 等待 DNS 生效（10-30 分钟）

4. 访问验证：
   ```
   https://www.gdkunyuan.cn
   ```

#### 保留 Vercel 作为备份

Vercel 项目保留，可以：
- 作为海外访问备份
- 测试环境
- 应急切换

## 常见问题

### Q1: 备案审核不通过怎么办？

**A:**
1. 查看审核不通过原因
2. 按要求修改资料
3. 重新提交审核

### Q2: 备案需要多长时间？

**A:**
- 腾讯云初审：1-2 天
- 管局审核：7-20 天
- 总计：7-20 天

### Q3: 备案期间网站能访问吗？

**A:**
- 中国大陆：❌ 不能访问
- 海外：✅ 可以访问（如果配置了）
- Vercel：✅ 可以访问（无需备案）

### Q4: 备案通过后如何切换到 CDN？

**A:**
1. 在腾讯云 CDN 添加域名
2. 修改 DNS 解析
3. 等待 DNS 生效
4. 验证访问
